This page shows printers that will work without any additional software because they support the [AirPrint™](https://support.apple.com/en-us/HT201311) and/or [IPP Everywhere™](https://www.pwg.org/ipp/everywhere.html) standards for driverless printers. These printers also often support [Mopria®](https://mopria.org/certified-products) as used on Android OS and Microsoft Windows®, and [Wi-Fi Direct Print Services](https://www.wi-fi.org/) for printing directly via Wi-Fi.

## What is Driverless Printing?

Driverless printing means that no printer-specific software or driver needs to be installed on your computer. The printer advertises its capabilities using standard network protocols, and your operating system can communicate with it directly.

Modern printers that support driverless printing use one or more of these technologies:

- **AirPrint™** — Apple's standard, widely adopted by printer manufacturers for seamless printing from macOS, iOS, and iPadOS
- **IPP Everywhere™** — The Printer Working Group's (PWG) vendor-neutral standard for driverless printing on any operating system
- **Mopria®** — A widely supported standard for printing from Android and Windows devices
- **Wi-Fi Direct Print Services** — Allows printing directly over Wi-Fi without a network router

## Finding a Driverless Printer

The OpenPrinting printer database provides a searchable list of all known printers and their Linux/Unix compatibility. For driverless printers specifically, you can browse the live database at [openprinting.github.io/printers](https://openprinting.github.io/printers/).

## Legacy Printer Support

If you have an older printer that does not support driverless printing, you may still be able to use it with a free software printer driver. The [Foomatic](https://openprinting.org/printers) database lists thousands of printers supported by free software drivers.

Alternatively, [Printer Applications](/drivers/) can revive older printers — even under modern operating systems like current versions of Windows using the [Windows Subsystem for Linux](/wsl-printer-app/).
